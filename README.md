## Project Setup

1. `git clone https://github.com/Offset-official/shortlist`
2. Make sure that `.env` exists at the same level as `.template.env` (Find on Secret Project WhatsApp Group)
3. To install the dependencies- `pnpm i`

## Run

```bash 
pnpm dev 
```
